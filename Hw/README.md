Это приложение на С++ для управления библиотекой. Проект показывает работу с классами и файловой системой

cmake version 4.2.1
Apple clang version 17.0.0

Функционал: 
Добавление и просмотр книг, регистрация пользователей,выдача и сдача книг и сохранение всего этого в файл

Сборка
mkdir build
cd build
cmake ..
make
./library
